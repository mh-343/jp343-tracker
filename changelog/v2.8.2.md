## Features

- **Update notification**: When the extension updates while you're watching, a banner prompts you to reload so tracking can resume.
- **Dashboard tab reuse**: The popup's Dashboard and Settings buttons now switch to an existing dashboard tab instead of opening a new one.

## Fixes

- **Stretch goal calculation**: Fixed the "past the final stretch" tooltip and overflow bar using the wrong threshold (2.5x instead of 3.0x).

## Improvements

- **Welcome page hint**: Added a note to reload already-open streaming tabs after first install.
- **YouTube context recovery**: Simplified redundant context-invalidation checks across the content script.
