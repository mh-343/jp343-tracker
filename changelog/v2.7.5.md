## Features

- **Target start time**: Set a target time for each weekday when you want to start immersing. A 14-day chart shows how early or late you actually started, with average and median stats.
- **Collapsible dashboard cards**: Click any card title to collapse it. Your preferences are remembered across sessions.

## Fixes

- **Netflix series detection**: More reliable series ID matching for Japanese titles.
- **YouTube channel recovery**: Faster channel ID detection after navigating between videos.

## Improvements

- **Dashboard UI clarity**: Clearer tooltips, better error messages, consistent terminology across the dashboard.
