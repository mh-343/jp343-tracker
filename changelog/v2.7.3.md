## Fixes

- **Netflix project detection**: Sessions for Japanese titles are now correctly assigned to separate projects instead of merging into one.
- **YouTube channel detection**: Switching videos within the same tab no longer assigns the session to the previous channel.
- **Channel list dedup**: Duplicate entries from mixed handle/@-ID formats are now merged automatically.

## Features

- **Daily goal sync**: Your daily goal setting now syncs between the extension and jp343.com.

## Improvements

- **Goal editor**: Enter key saves, click handling no longer closes the editor unexpectedly, input validation shows feedback for out-of-range values.
