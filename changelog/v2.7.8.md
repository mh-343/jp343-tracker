## Features

- **Popup session history**: Synced sessions from the server now appear in the popup alongside local pending entries.

## Fixes

- **Goal bar accuracy**: Daily stats no longer drift when multiple sessions save at the same time.
- **Netflix ad titles**: Placeholder text like "after the ads" no longer gets cached as a video title.
- **Netflix video navigation**: Switching between videos now correctly ends the previous session.

## Improvements

- **Sync reliability**: All server requests now use timeouts and large backlogs sync in smaller batches.
