## Features

- **ttu ebook reader**: Reading time from the ttu ebook reader (reader.ttsu.app) now counts as immersion, a second reading source next to Mokuro. Opt-in, enable it in the settings.
- **Custom sites (beta)**: Add your own sites to auto-track video on players that are not built in yet.
- **Rename custom site series**: Custom site sessions start with a raw label from the address. Rename a session once and every future episode of that series uses your name, on this device and in your account history.
- **Keyboard shortcuts**: Start and stop tracking with keyboard shortcuts.

## Improvements

- **Settings sidebar**: Dashboard settings are grouped into a sidebar layout instead of one long scroll, with cleaner toggles in light mode.
- **Vote chip**: The difficulty vote chip now appears during your first session on a channel, after about a minute of watching.

## Fixes

- **Japanese-only filter**: Allowed channels stay visible on channel pages when the Japanese-only filter is on.
- **Settings and filter after updates**: Settings toggles respond quickly again and the YouTube filter no longer goes stale after an extension update.

## Notes

- **License**: Upgraded from GPL-3.0 to AGPL-3.0, the strongest copyleft license. The extension stays fully open source; anyone building on the code must keep their version open under the same license. The code is public on GitHub so you can read, audit and build it yourself.
