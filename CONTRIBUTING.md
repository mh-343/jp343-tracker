# Contributing

Thanks for wanting to help.

- For anything bigger than a small fix, open an issue first so I can say yes
  before you invest time.
- By opening a pull request you license your contribution under the project
  license (AGPL-3.0) and additionally grant the project owner the right to
  use, modify, distribute and relicense it as part of this project, and you
  confirm the contribution is your own work.

Follow the conventions of the surrounding code (naming, file size limits,
no `any`, minimal English comments) and run `npm run check` before
submitting.
