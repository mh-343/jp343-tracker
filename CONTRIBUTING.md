# Contributing

Thanks for wanting to help. Two things to know before you start, since this
project is source-available rather than open source:

- For anything bigger than a small fix, open an issue first so I can say yes
  before you invest time.
- The [LICENSE](LICENSE) explicitly allows making changes for the sole
  purpose of proposing them here as a pull request. By opening a pull request
  you agree that your contribution may be used, modified, distributed and
  relicensed by the project owner as part of this project, and you confirm
  the contribution is your own work.

Follow the conventions of the surrounding code (naming, file size limits,
no `any`, minimal English comments) and run `npm run check` before
submitting.
